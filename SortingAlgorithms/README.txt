
For MergeSort execute by using the following commandline argument 

gcc -o mergeSort mergeSort.c sortDriver.c metrics.c

You can now run the program with the command:

.\mergeSort, Press Enter, then type the numbers you want to sort then Press Ctrl + D 


For InsertSort execute by using the following commandline argument 

gcc -o mergeSort mergeSort.c sortDriver.c metrics.c

.\insertSort,Press Enter, then type the numbers you want to sort then Press Ctrl + D
